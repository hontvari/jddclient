How to install Jddclient

Follow the instructions corresponding to your operating system below. After installation you have to configure jddclient.

Ubuntu (Debian etc.)

Simply double click on the .deb file. Or download the file, and in the terminal enter
sudo gdebi <FILE NAME>

Windows

  1. Download the binary distribution and unzip it.
  2. Move the configuration files into the %ALLUSERSPROFILE%\jddclient directory. Instead of this you can use the --configuration <path-to-jddclient.conf.xml> option in the command which starts jddclient (see below).
  3. Configure it as a service (for example using Java Service Wrapper) or add as a scheduled task. Use this command in a service configuration:
    java -Dlogback.configurationFile=<path-to-logback-daemon.xml> -jar <path-to-jddclient.jar> --daemon
    Use this command in a scheduled task:
    java -Dlogback.configurationFile=<path-to-logback-daemon.xml> -jar <path-to-jddclient.jar>

Other *nix operating systems

  1. Download the binary distribution and unzip it.
  2. Move the configuration files to the /etc/jddclient directory. Instead of this you can use the --configuration <path-to-jddclient.conf.xml> option in the command which starts jddclient (see below).
  3. Configure a daemon or add a cron job. For an example see the Ubuntu/Debian package configuration files and the instructions for Windows above.
