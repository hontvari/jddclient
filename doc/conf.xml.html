Detectors

Router

The class name which must be specified: jddclient.detector.Router

The Router detector downloads the status page from a home/small business router and extracts the IP address from that HTML page.

See the comments in the default file for more information on the child elements of this detector.

Interface

The class name which must be specified: jddclient.detector.Interface

The Interface detector queries the IP addresses assigned to a local interface (network card).

Child elements:
  • name: the name of the interface, for example eth0
